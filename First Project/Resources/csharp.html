C#

C# (pronounced "C Sharp") is Microsoft's modern, object-oriented programming language. As a cornerstone of the .NET ecosystem, it combines the power of C++ with the simplicity of Visual Basic. C# is versatile enough for Windows apps, web services, games (via Unity), and cross-platform development with .NET Core.

Key Features

  • Object-Oriented: Full support for encapsulation, inheritance, and polymorphism
  • Type-Safe: Strong typing with compile-time checking
  • Modern Syntax: Features like LINQ, async/await, and pattern matching
  • Memory Management: Automatic garbage collection
  • Cross-Platform: Runs on Windows, Linux, and macOS via .NET Core
  • Rich Standard Library: Comprehensive .NET Base Class Library
  • Language Interoperability: Works seamlessly with other .NET languages
  • Versatile: Used for web, mobile, desktop, games, and cloud development

.NET Platform Architecture

The .NET platform provides a comprehensive environment for C# development:

  • Common Language Runtime (CLR): Execution engine with JIT compilation
  • Base Class Library (BCL): Fundamental classes for common tasks
  • ASP.NET Core: Framework for building web applications
  • Entity Framework: ORM for database access
  • Xamarin: For cross-platform mobile development

Where C# is Used

  • Enterprise Software: Business applications, CRM, ERP systems
  • Game Development: Unity game engine's primary scripting language
  • Web Development: ASP.NET Core for backend services
  • Desktop Applications: Windows Forms, WPF, UWP

Popular C# Frameworks & Libraries

Object-Oriented Programming

public class Vehicle
{
public string Make { get; set; }
public string Model { get; set; }

public virtual void DisplayInfo()
{
Console.WriteLine($"Make: {Make}, Model: {Model}");
}
}

public class Car : Vehicle
{
public int Doors { get; set; }

public override void DisplayInfo()
{
base.DisplayInfo();
Console.WriteLine($"Doors: {Doors}");
}
}

LINQ (Language Integrated Query)

using System;
using System.Linq;

class Program
{
static void Main()
{
int[] numbers = { 5, 4, 1, 3, 9, 8, 6, 7, 2, 0 };

// LINQ query to find even numbers
var evenNumbers = from num in numbers
where num % 2 == 0
select num;

Console.WriteLine("Even numbers:");
foreach (var num in evenNumbers)
{
Console.WriteLine(num);
}
}
}

C# Version History

  • 2002: C# 1.0 with .NET Framework 1.0
  • 2005: C# 2.0 (generics, nullable types)
  • 2007: C# 3.0 (LINQ, lambda expressions)
  • 2010: C# 4.0 (dynamic typing, named arguments)
  • 2012: C# 5.0 (async/await)
  • 2015: C# 6.0 (null-conditional operator, string interpolation)
  • 2017: C# 7.0 (tuples, pattern matching)
  • 2019: C# 8.0 (nullable reference types, ranges)
  • 2020: C# 9.0 (records, init-only properties)
  • 2021: C# 10.0 (file-scoped namespaces, global using)
  • 2022: C# 11.0 (raw string literals, required members)
