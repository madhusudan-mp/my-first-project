HTML

HTML (HyperText Markup Language) is the foundational language of the web. It provides the structure and content of every webpage, using tags to define elements like headings, paragraphs, images, and links. While not a programming language, HTML works with CSS for styling and JavaScript for interactivity to create modern websites.

Think of HTML as the skeleton of a webpage - it defines the structure, while CSS adds the skin and JavaScript provides the muscles.

History of HTML

1989

Tim Berners-Lee invents HTML while working at CERN as a way to share scientific documents.

1991

First public description of HTML tags is published.

1995

HTML 2.0 is published as the first standard HTML specification.

1997

HTML 3.2 is released, adding tables, applets, and text flow around images.

1999

HTML 4.01 becomes a W3C Recommendation, introducing stylesheets and scripting.

2000

XHTML 1.0 is introduced as a reformulation of HTML 4 in XML.

2008

HTML5 draft is first published by WHATWG.

2014

HTML5 is completed and standardized, bringing new semantic elements and APIs.

Key Features of HTML

  • Markup Language: Uses tags to define elements and their purpose
  • Platform Independent: Works across all operating systems and devices
  • Hypertext: Allows linking between documents via hyperlinks
  • Semantic Structure: Provides meaning to content through semantic elements
  • Media Support: Can embed images, video, audio, and other multimedia
  • Forms: Enables user input and data collection
  • Accessibility: Supports features to make content accessible to all users
  • Extensible: Can be combined with CSS and JavaScript for enhanced functionality

Basic HTML Document Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

Common HTML Elements

Element Description Example
<h1> to <h6> Heading elements (h1 is highest level) <h1>Main Title</h1>
<p> Paragraph <p>This is a paragraph.</p>
<a> Anchor (hyperlink) <a href="https://example.com">Link</a>
<img> Image <img src="image.jpg" alt="Description">
<div> Division or section <div>Content group</div>
<span> Inline container <span>Styled text</span>
<ul>, <ol>, <li> Unordered list, ordered list, list item <ul><li>Item</li></ul>
<table>, <tr>, <td> Table, table row, table data <table><tr><td>Data</td></tr></table>
<form>, <input> Form and input elements <form><input type="text"></form>

HTML5 Semantic Elements

HTML5 introduced new semantic elements that better describe their meaning:

  • <header> - Container for introductory content
  • <nav> - Navigation links
  • <main> - Main content of the document
  • <article> - Independent, self-contained content
  • <section> - Thematic grouping of content
  • <aside> - Content aside from the main content
  • <footer> - Footer for a document or section
  • <figure> and <figcaption> - Figures with captions
  • <time> - Machine-readable date/time
  • <mark> - Highlighted text
